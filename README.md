# 🐔 Unified Farm BLM – Poultry Farming Calculator

A modern, responsive, and intelligent poultry farming calculator built for farmers in Africa and worldwide.  
This tool helps you calculate **feed requirements, total costs, mortality, revenue, and profit** for multiple bird types — all in real time.

Designed and developed by **Unified Farm BLM**.

---

## 🌟 Features

### ✔ Live Auto‑Update Calculations  
Results update instantly as the farmer types — no “Calculate” button needed.

### ✔ Ease‑Out Number Animation  
Smooth, premium number transitions (0 → 5000) for a modern experience.

### ✔ Auto‑Save System  
Inputs are saved automatically and restored when the farmer returns.

### ✔ Smart Reset  
Inputs reset only when the bird type changes.  
Language and currency changes keep the data.

### ✔ Multi‑Bird Support  
- Broilers  
- Layers  
- Kuroilers  
- Sasso  
- Ducks  
- Turkeys  
- Guinea Fowl  
- Quails  

### ✔ Multi‑Language Ready  
English + Luganda (easily expandable).

### ✔ Multi‑Currency Ready  
USD, UGX, KES, AED, and more.

### ✔ Premium UI  
- Textured brown headers  
- Soft gold medium‑weight text  
- Sharp corners  
- Light/Dark mode  
- Smooth collapsible sections  
- Drawer menus for bird, language, and currency  

---

## 🧮 What the Calculator Computes

- Feed intake (based on FCR × weight)  
- Total feed cost  
- Chick cost  
- Vaccination cost  
- Housing cost  
- Labor cost  
- Mortality adjustment  
- Survivors  
- Total cost  
- Revenue  
- Net profit  

All calculations are performed **live** using JavaScript.
